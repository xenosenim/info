‘Khīṇā jāti, vusitaṃ brahmacariyaṃ, kataṃ karaṇīyaṃ, n·āparaṃ itthattāyā’ti.

‘Birth is ended, the brahmic life has been lived, what was to be done has been done, there is nothing more for this existence’.

‘Khīṇā jāti,

‘Birth is ended,
vusitaṃ brahmacariyaṃ,

the brahmic life has been lived,
kataṃ karaṇīyaṃ,

what was to be done has been done,
n·āparaṃ itthattāyā’ti.

there is nothing more for this existence’.
