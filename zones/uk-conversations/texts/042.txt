# екскурсія до міста

## экскурсия по городу

|чи відкритий pинок щонеділі?||рынок работает по воскресеньям?
|чи відкритий ярмарок щопонеділка||ярмарка работает по понедельникам?
|чи відкрита виставка щовівтірка?||выставка работает по вторникам?
|чи відкритий зоопарк щосереди?||зоопарк работает в среду?
|чи відкритий музей щочетверга?||музей по четвергам открыт?
|чи відкрита галерея щоп’ятниці?||галерея открыта по пятницам?
|чи можна фотографувати?||фотографировать можно?
|чи потрібно платити за вхід?||вход платный?
|скільки коштує вхід?||сколько стоит вход?
|чи є знижка для груп?||для групп есть скидка?
|чи є знижка для дітей?||для детей есть скидка?
|чи є знижка для студентів?||для студентов есть скидка?
|що це за будівля?||что это за здание?
|скільки років цій будівлі?||сколько этому зданию лет?
|хто побудував цю будівлю?||кто построил это здание?
|я цікавлюся архітектурою.||я интересуюсь архитектурой.
|я цікавлюся мистецтвом.||я интересуюсь искусством.
|я цікавлюся живописом.||я интересуюсь живописью.
